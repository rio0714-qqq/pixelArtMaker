## 概要
### *ドット絵作成アプリ(Android)*
ドット絵が趣味なので自分用に作成。<br>
UI/UXはレトロな雰囲気を意識。<br>
今後はSQLite等を用いたアプリ内作品保存機能の追加を検討中。<br>

### *機能*
- 一般的な16×16, 24×24、32×32、48×48ピクセルの他、自分でサイズを指定してドット絵を作成することが可能
- ドット描画の際のガイド線は表示/非表示選択可能
- カラーコード、RGB値のどちらかを指定してパレット作成ができ、パレットから色を選んで描画できる
- 作成したドット絵は端末に保存できる
